Interactive Resume

Github: https://github.com/stokarz1/resume_jquery

This is an interactive resume I created to polish up my jQuery skills and get familiar with Bootstrap 4.

This site loads all of my employment information from a json file via Ajax and and then does some cloning and Dom writing to display all the jobs, taks, and technology used at each task.

Click on the bulleted task below any Employer to view what technologies I used to perform tht taksk. You can also enter any technology name in the search box in the upper right and I'll redraw the employment section with any tasks that use the technology you're searching for already expanded and the full name of the technology highlighted. The search requires 3 characters minimum and is case insensitive.

If you don't want to use this interactive version of my resume, you can download a pdf version here.

Project Demo: Coca-Cola Freestyle GUI Proposal

Github: https://github.com/stokarz1/freestyle_demo

This is a demo I created to show my entire workflow for a new project. I wanted to propose a new GUI for the Coca-Cola Freestyle machines to make them more intuitive to use.

For the first step of any project, I create a wireframe. To download a pdf of the wireframe for this project click here.

I wanted to design an interface that would confuse people less and let them get exactly what they want in a more timely manner. To do this, I would use existing conventions that people already understand from the internet, and also develop a clear language of icons that would be consistantly used throughout the application. I'd use shadows and shading to represent state, like a button being pressed, instead of arbitrary icon sizing.
